# New Tech - Frontend Ecommerce Website

A responsive, frontend-only ecommerce web project built as a university assignment.

New Tech showcases a complete online shopping user interface for electronics, including home, product browsing, product details, cart/wishlist, account pages, and support pages.

## Project Type

- Academic project (University assignment)
- Frontend only (no backend/API/database integration)

## Features

- Responsive layout for desktop and mobile screens
- Modern navigation bar with search and quick-access icons
- Home page hero section with interactive carousel
- Product listing page with:
  - Product cards
  - Pagination UI
  - Sort options
  - Filter sidebar
- Product detail view with image slider and purchase actions
- Wishlist and cart UI pages
- Account-related pages:
  - Profile settings
  - Orders
  - Payments
  - Reviews
  - Returns/Cancellations
- Additional pages:
  - Contact
  - Help center
  - Message center
  - Seller login
  - Admin panel
- Custom styling and JavaScript interactions

## Tech Stack

- HTML5
- CSS3
- Bootstrap
- JavaScript (Vanilla JS)
- Bootstrap Icons (CDN)

## Getting Started

1. Clone or download this repository.
2. Place the project folder inside your web server directory (for example, XAMPP `htdocs`).
3. Open `index.html` in a browser.

You can also run it directly by double-clicking `index.html`, but using a local server is recommended for a smoother development workflow.

## Notes

- This project is built for demonstration and UI/UX practice.
- Since there is no backend, actions like sign in, checkout, order processing, and data persistence are mock/front-end only.
